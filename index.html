▲ Vercel

Deploy ke Vercel

Masukkan nama proyek dan konten HTML, lalu dapatkan instruksi deploy siap pakai.

📖 Cara Deploy
  1. Generate file di atas, lalu copy isi index.html
  2. Buat folder baru di komputer kamu
  3. Simpan sebagai index.html
  4. Pergi ke vercel.com → Login / Daftar
  5. Klik Add New → Project
  6. Upload folder atau sambungkan GitHub
  7. Klik Deploy — selesai! 🎉
